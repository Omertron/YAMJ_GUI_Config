    <div class="taxonomy"><ul class="links inline"><li class="taxonomy_term_3 first"><a href="/pch/taxonomy/term/3" rel="tag" title="Anything else. Enough of these and a new category will be created">Other</a></li>
<li class="taxonomy_term_21"><a href="/pch/taxonomy/term/21" rel="tag" title="">YAMJ</a></li>
<li class="taxonomy_term_47"><a href="/pch/taxonomy/term/47" rel="tag" title="">Config</a></li>
<li class="taxonomy_term_46 last"><a href="/pch/taxonomy/term/46" rel="tag" title="">Install</a></li>
</ul></div>
    <div class="content"><p>This program is designed to make configuring YAMJ a little easier for new users.</p>
<p>&nbsp;Latest Version: 0.3</p>
<p>&nbsp;<img src="/pch/system/files/GUI Config v0_2.png" /></p>
<p>To use it, run the exe file after unpacking it from the zip file.</p>
<p>Click the &quot;Select Dir&quot; button to tell the program where your video files are located.</p>
<p>Choose the NMT Path that is relevant for your Video location.</p>
<p>If you select &quot;Network&nbsp;Path&quot; a pop-up will appear asking for the IP&nbsp;Address of your NMT, it will then locate the shares that you have on your NMT and populate them in the network shares section.</p>
<p>Once you have done that, select Add and then Save and the program will ask you where you want the files save.&nbsp;WARNING:&nbsp;It will overwrite these files if they exist.<br />
The files it creates are:</p>
<ul>
<li>My_Library.xml</li>
<li>My_YAMJ.cmd</li>
</ul>
<p>When you are ready to run YAMJ, simply browse to the folder and double click the &quot;My_YAMJ.cmd&quot; file.</p>
<p>Thanks to Dizingof for his first YAMJ GUI&nbsp;and Legion455 for his scraper of the NMT share code.</p>
<p>Please post any comments, questions, reports or requests on this thread <a href="http://www.networkedmediatank.com/showthread.php?tid=11819" target="_blank">*HERE*</a></p>
